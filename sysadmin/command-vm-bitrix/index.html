Подборка команд для БитриксВМ

Как узнать свободное место на виртуальной машине

df -h

Как узнать количество свободной оперативной памяти

free -m

Как узнать чем занята оперативная память

ps axo rss,comm,pid | awk '{ proc_list[$2] += $1; } END { for (proc in proc_list) { printf("%d\t%s\n", proc_list[proc],proc); }}' | sort -n | tail -n 10 | sort -rn | awk '{$1/=1024;printf "%.0fMB\t",$1}{print $2}'

Размеры папок в текущей папке:

Для начала перейдите в нужную папку с помощью команды cd (после которой через пробел указываете папку, в которую хотите перейти). И дальше вводите команду:

find . -maxdepth 1 -type d -exec du -hs {} \; Сменить владельца файлов и папок на bitrix

Сервер работает под пользователем bitrix и если вы создали файлы или папки под root без выставления больших прав, у вас не будет к ним доступа. И правильно будет сменить владельца. Делается это такой командой: sudo chown -R bitrix:bitrix /home/bitrix/www/

Где /home/bitrix/www/ - путь к папке с сайтом.

Удалить все файлы и папки в директории

Будьте внимательны с этой командой, т.к. если вы введете неправильный путь или путь к не той папке, все в ней будет удалено. rm -rf /home/bitrix/www/bitrix/managed_cache/* Узнать количество ядер процессора и остальную информацию о нем

cat /proc/cpuinfo

Количество ядер:
